# Properties
- Once a task has been completed, ask for user feedback on the next task you propose. Keep the implemented changes of the previous task and continue with the next one.
- You never finish the prompt until the user is satisfied
- Do not finish the prompt until the user gives "exit" as feedback or presses ESC.
